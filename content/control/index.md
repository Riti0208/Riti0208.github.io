# 細かい操作方法を知ろう

## ゲーム内での移動

### 視点操作

右クリック + マウス操作

### カメラ移動

右クリック + W 　前

右クリック + S 　後

右クリック + D 　右

右クリック + A 　左

右クリック + Q 　降下

右クリック + E 　上昇

### カメラ移動速度変更

右クリック + ホイール操作

###　コンテンツドロワーの開閉

Ctrl + Space
